# Add es

Adds an “es” if n is \> 1 and reverse is false (default) Adds “es” if n
is == 1 and reverse is true

## Usage

``` r
.es(n, reverse = FALSE)
```

## Arguments

- n:

  the number of elements

- reverse:

  logical, defaults to FALSE (see above)

## Value

a string

## Details

e.g. match and matches
