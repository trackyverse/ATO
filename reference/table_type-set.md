# Assign a new table type to the ATO

Assign a new table type to the ATO

## Usage

``` r
table_type(x) <- value

# S4 method for class 'ATO'
table_type(x) <- value
```

## Arguments

- x:

  an [`ATO`](https://trackyverse.github.io/ATO/reference/ATO_package.md)

- value:

  The new table type to be used inside the ATO.

## Value

Nothing, acts directly on x
