# Add s

Adds an “s” if n is \> 1 and reverse is false (default) Adds “s” if n is
== 1 and reverse is true

## Usage

``` r
.s(n, reverse = FALSE)
```

## Arguments

- n:

  the number of elements

- reverse:

  logical, defaults to FALSE (see above)

## Value

a string

## Details

e.g. detection and detections
