# S4 class: ATO_tbl

An S4 class for the
[`ATO`](https://trackyverse.github.io/ATO/reference/ATO_package.md)
table format (@tbl). Does not contain internal slots. It is a single
value. One of: "data.frame", "data.table", or "tibble"

## Usage

``` r
.ATO_tbl
```

## Format

An object of class `ATO_tbl` (inherits from `character`) of length 1.

## Details

The prototype @tbl slot contains the default value for the @log slot.
