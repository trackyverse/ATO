# Match transmitters in the detections to the target tags

Automatically called by the
[`add`](https://trackyverse.github.io/ATO/reference/add.md) functions.

## Usage

``` r
.match_det_tag_base(x, silent = FALSE)
```

## Arguments

- x:

  an [`ATO`](https://trackyverse.github.io/ATO/reference/ATO_package.md)

- silent:

  Supresses summary messages

## Value

the updated ATO
