# Match transmitters in the observations to the target tags

Automatically called by the
[`add`](https://trackyverse.github.io/ATO/reference/add.md) functions.

## Usage

``` r
.match_obs_tag(x, silent = FALSE)
```

## Arguments

- x:

  an [`ATO`](https://trackyverse.github.io/ATO/reference/ATO.md)

- silent:

  Supresses summary messages

## Value

the updated ATO
