# S4 class: ATO

The Animal Tracking Object (ATO) class contains slots that allow
researchers to package their study data in a standard format, to be used
by the R packages that collectively make the trackyverse.

## Slots

- `det`:

  The detections slot. See
  [`ATO_det`](https://trackyverse.github.io/ATO/reference/ATO_det.md).

- `dep`:

  The deployments slot. See
  [`ATO_dep`](https://trackyverse.github.io/ATO/reference/ATO_dep.md).

- `tag`:

  The tags slot. See
  [`ATO_tag`](https://trackyverse.github.io/ATO/reference/ATO_tag.md).

- `ani`:

  The animals slot. See
  [`ATO_ani`](https://trackyverse.github.io/ATO/reference/ATO_ani.md).

- `obs`:

  The observations slot. See
  [`ATO_obs`](https://trackyverse.github.io/ATO/reference/ATO_obs.md).

- `log`:

  The log slot. See
  [`ATO_log`](https://trackyverse.github.io/ATO/reference/ATO_log.md).

- `tbl`:

  The table style slot. See
  [`ATO_tbl`](https://trackyverse.github.io/ATO/reference/ATO_tbl.md).

- `pkg`:

  A list slot reserved for package outputs.
